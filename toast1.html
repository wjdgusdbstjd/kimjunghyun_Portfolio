TOAST1

토스트에 대해 소개하는 CSS 웹사이트

  • HTML
  • CSS

  • 프로젝트에서 나의 역할

    기획부터 디자인, 구현까지를 모두 담당하였습니다. 주제를 정해 기획서를 작성하였고, 기획서를 토대로 와이어 프레임과 1차 디자인 시안 작업을 진행하여 웹을 구현하였습니다.

  • 프로젝트 진행 과정 중 어려움과 해결

    css만을 이용해 웹사이트를 제작하였기 때문에 동적인 인터렉션을 보여주는데 어려움이 있었습니다. 그래서 아예 스크롤이 거의 없이 단일 페이지에 정보를 담고자 구성하였고, hover를 활용해 변화를 보여주었습니다. 또한 크롬에서는 정상적으로 보여지던 페이지가 사파리 브라우저에서는 몇몇 텍스트의 배치가 깨지는 현상이 발생하여 이를 텍스트 크기 변경을 통해 수정하였고, 이후 폰트 설정으로 인해 이러한 문제가 발생하였음을 확인하였습니다.

  • 프로젝트를 통한 발전

    처음 반응형으로 제작한 웹페이지로 이를 통해 반응형웹의 중요성과 어떤식으로 구현되는지를 배웠지만, 실제로 구현하는 방식에 아쉬움을 느껴 더욱 공부하게 된 계기가 되었습니다. 브라우저마다 웹이 조금씩 다르게 보인다는 사실과 사파리 브라우저에서 깨지지않으면 거의 모든 브라우저 상에서 동일하게 보인다는 것을 알게되었습니다.
