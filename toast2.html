TOAST2

토스트를 주제로 한 인터렉티브 웹사이트

  • HTML
  • CSS
  • JavaScript
  • Jquery

  • 프로젝트에서 나의 역할

    기획 단계에서 자료조사와 리서치를, 디자인 단계에서 앱의 세부적인 디자인을 함께 진행하였습니다. 웹 개발의 단계에서는 디자인되어있는 웹의 구성을 보고 html,css,javascript를 활용해 실제 웹사이트로 구현하였습니다.

  • 프로젝트 진행 과정 중 어려움과 해결

    앱을 설명하는 페이지에서의 스크롤방식을 어떻게 구현해야하는지 알아내는 과정에서 어려움을 겪었습니다. 처음엔 구글과 Chat GTP등의 경로를 통해 웹페이지의 스크롤 방식에 대해 먼저 알아보고, 구현하고자하는 스크롤방식을 활용한 웹사이트들을 찾아 개발자도구에서 어떤 코드들을 사용했고, 어떤 플러그인을 활용했는지 꼼꼼히 분석하여 원하는 스크롤 방식을 구현해냈습니다.

  • 프로젝트를 통한 발전

    플러그인에 대한 개념을 공부했습니다. 플러그인 내 코드를 어떻게 수정해야하는지, 동일한 플러그인을 두개 이상 사용하고자하려면 어떻게 해야하는지에 대해 배웠고, 웹페이지를 만들면서 반응형에 대해 아쉬움이 많아 따로 공부해보면서 반응형 단위와 사용성에 대해 고민해보는 계기가 되었습니다.
